Projet de Programmation Orientée Objet – Java

Ce projet a été réalisé dans le cadre de l’enseignement de Programmation Orientée Objet en Java (L2). Il consiste à développer une application permettant de gérer des fichiers MP3 et leurs métadonnées.

Objectifs du projet

  • Explorer récursivement un répertoire pour trouver des fichiers MP3
  • Extraire les métadonnées (titre, artiste, durée, album, etc.)
  • Créer et sauvegarder des playlists dans différents formats
  • Proposer un mode console (CLI) et un mode graphique (GUI)

Technologies utilisées

  • Java
  • Programmation Orientée Objet (classes, interfaces, UML)
  • Librairie mp3agic pour les métadonnées
  • Interface graphique avec Swing

Résultat

L’application permet d’extraire et d’afficher les métadonnées des fichiers MP3, de générer des playlists (XSPF, M3U8, JSPF) et d’interagir avec l’utilisateur via une interface en ligne de commande ou graphique.

Projet Snaketris C++

Projet réalisé en C++ consistant à créer un jeu combinant les mécaniques de Snake et de Tetris, réduisant la taille du serpent lors de l'alignement de case de même couleur.

Caractéristiques :

  • Développement en C++
  • Compilation avec Makefile
  • Gestion graphique simple
  • Gestion des collisions et du score

Ce projet nous a permis de renforcer nos compétences en programmation, en organisation de projet avec Makefile et en logique de jeu.
